# 更新日志

## 2.0.2 - 2026-06-23

- 修复 OxygenOS / Oplus 通知中心在两条同应用通知聚合摘要场景下，规则图标可能被系统装饰层覆盖成纯色块的问题。
- 对 Oplus 聚合摘要图标使用紧凑遮罩渲染，并在系统异步重绘后重新应用透明背景和规则图标，避免图标与普通通知割裂。
- 移除临时诊断日志和视图树打印，保留必要的异常告警，降低 SystemUI 日志噪声。
- 使用公开通知字段识别媒体通知，避免通过隐藏 API 反射判断媒体样式。
- 使用 AndroidX 注册 SystemUI 配置刷新广播，减少 SDK 版本分支和 lint 告警。
- 补齐 Android 12+ 数据提取规则和旧版备份规则，继续保持应用数据不参与系统备份。
- 清理 lint 告警并记录少量有意保留的项目约束，`lintDebug` 已通过且无 warning。
